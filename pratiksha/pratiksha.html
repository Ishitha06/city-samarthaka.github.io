Healthcare

Smart City Healthcare Management System

A smart-city healthcare management system designed for a population of approximately 2.6 million. It uses syllabus-based data structures like Hash Tables, Heaps, BFS, Graphs, and AVL Trees to ensure efficient routing, patient search, emergency handling, and hospital directory management.

SDGs: 3, 9, 11

Objectives

  • Fast access to patient medical records
  • Smart emergency handling with priority scheduling
  • Efficient ambulance routing using BFS
